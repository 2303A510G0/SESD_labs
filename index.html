3

Create Feature Branches

üîß Merge Conflict Resolution

When merge conflicts occur, Git marks the conflicting sections in your files. You'll need to:

  1. Open the conflicted file and locate the conflict markers (<<<<<<<, =======, >>>>>>>)
  2. Decide which changes to keep or combine both versions
  3. Remove the conflict markers
  4. Stage the resolved file with git add
  5. Complete the merge with git commit
